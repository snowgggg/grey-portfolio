# Grey Portfolio

Dark cinematic personal portfolio for Grey, a Web3 product designer. Built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Motion

## Assets

The hero reel is served from `public/assets/hero/hero-video.mp4`.
